# Security

Safety is the most important part of this guide.

Never share:

- seed phrase;
- private key;
- wallet export;
- `kms.json`;
- identity backup files;
- browser profile data;
- full screenshots with sensitive wallet/account data.

Use a separate wallet or identity key for agent activity.

Do not use your main wallet with valuable assets.

If any terminal command, website, Discord DM, script, or AI agent asks for a seed phrase or private key:

**STOP immediately.**

## Paid claim warning

Default mode is **FREE CLAIM ONLY**.

If you wanted a free claim but see:

- `10 USDC`;
- `amount=10000000`;
- any amount greater than `0`;

then stop. This is not a free claim.
