# Quick Start

For the shortest safe path, open:

[COMMAND_CENTER.md](./COMMAND_CENTER.md)

Then run Windows Agent Doctor.

The terminal will tell you what to do next.

---

## Main rule

Do not claim if the payment is not free.

Stop if you see:

```text
10 USDC
amount=10000000
amount > 0
```
