# After Claim Proof

After a successful Tile claim, save proof.

Do not rely only on memory or screenshots.

---

## Save this information

Copy this template:

```text
Agent name:
Agent folder:
Agent address:
DID:
Tile ID:
Canvas ID:
Coordinates:
Paid:
Amount:
Tx hash:
Claim date:
Explorer link:
Notes:
```

---

## Required success signs

A correct free claim should show:

```text
Paid: false
amount=0
Tile claimed / submitted successfully
```

If you see:

```text
Paid: true
amount=10000000
10 USDC
```

stop and review the logs.
