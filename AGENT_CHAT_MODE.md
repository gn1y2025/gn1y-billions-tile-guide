# AI Agent Chat Mode

This file contains instructions that a human can paste into an AI agent chat.

The agent must follow safety-first rules:

1. Never ask for seed phrases.
2. Never ask for private keys.
3. Never ask for wallet export files.
4. Never copy identity from another agent.
5. Never perform paid claims automatically.
6. Default mode is free claim only.
7. Stop if claim amount is greater than 0.
8. Stop if DID or agent address is wrong.
9. Ask the human before any critical action.

Full agent prompt will be added in the next update.
